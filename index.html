Clé API TMDB

Serie Manager utilise l'API TMDB pour rechercher des séries. Entrez votre propre clé API gratuite.

  1. Créez un compte sur themoviedb.org
  2. Allez dans Paramètres → API
  3. Copiez votre clé API (v3 auth)

Votre clé est stockée uniquement dans votre navigateur et n'est jamais envoyée à des tiers.

Configurer Google Drive

Pour sauvegarder votre CSV sur Google Drive, vous avez besoin d'un identifiant client OAuth 2.0 Google.

  1. Ouvrez Google Cloud Console → Credentials
  2. Créez un projet, activez l'API Google Drive
  3. Créez des identifiants : OAuth 2.0 → Application Web
  4. Ajoutez dans les origines JavaScript autorisées
  5. Copiez votre Client ID
